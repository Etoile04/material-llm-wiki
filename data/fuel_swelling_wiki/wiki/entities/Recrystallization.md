# 再结晶（Recrystallization）

> 金属核燃料中的辐照诱发再结晶（Irradiation-Induced Recrystallization）

## 定义

核燃料中的再结晶（Recrystallization, RX）是指晶粒在辐照作用下发生的结构重组：原始晶粒结构瓦解，新的细小等轴晶形成。在 U-Mo 和铀锆合金（U-Zr）燃料中，该过程由累积的裂变损伤驱动，并深刻影响裂变气体气泡（Fission Gas Bubble, FGB）行为。

## 物理机制

### 驱动力
- **储存能积累**：裂变诱发缺陷（空位、间隙原子、位错环）积累应变能
- **临界缺陷密度**：当缺陷密度超过阈值时启动再结晶（U-Mo 约 2–5 × 10²¹ fissions/m³）
- **温度依赖性**：阈值随温度升高而降低；高温下热回复（thermal recovery）与损伤积累相竞争

### 阶段
1. **孕育期（Incubation）**：缺陷在原始晶粒中积累（无微观结构变化）
2. **形核（Nucleation）**：新晶粒在高能位点（晶界、缺陷团簇）形成
3. **生长（Growth）**：新晶粒消耗原始晶粒结构
4. **完成（Completion）**：完全再结晶的细晶微观结构（~0.1–1 μm）

### 关键方程

**再结晶分数**（Rest 2005）：
```
f_rx = 1 - exp(-k × (Φ - Φ_0)^n)
```
其中：
- $Φ$ = 裂变密度（fissions/m³）
- $Φ_0$ = 孕育阈值（incubation threshold）
- $k$、$n$ = 材料相关常数

**晶粒尺寸演化**：
```
d_rx = d_0 × (1 - f_rx) + d_new × f_rx
```

## 对肿胀的影响

### 直接效应
- **晶粒细化** → 晶界面积增大 → 气泡形核位点增加
- **气泡超晶格（Gas Bubble Superlattice, GBS）溶解** → 晶界气泡快速释放气体
- **肿胀加速**：再结晶起始后肿胀率可提高 3–5 倍

### 间接效应
- **气体再分配**：从晶内（有序）转移至晶界（无序）
- **热导率退化**：晶界散射增强
- **力学性能变化**：软化、蠕变抗力降低

## 材料特异性行为

### U-10Mo
- 再结晶起始：~2–4 × 10²¹ fissions/m³（与温度相关）
- 原始晶粒尺寸：10–50 μm → 再结晶后：0.1–1 μm
- 再结晶过程中维持 γ 相稳定性（无相变）

### U-Zr（U-Pu-Zr）
- 因多相（α、β、γ、δ）结构更为复杂
- 相变可触发或抑制再结晶
- 成分再分配（Constituent Redistribution）影响局部再结晶行为

| 参数 | U-10Mo | U-Zr | 单位 |
