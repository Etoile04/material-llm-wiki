## 📐 公式推导

### 基本关系

**临界条件**:
```
T_critical = T_0 - A × (φ)^n
```

**参数**:
- T_critical: γ相稳定临界温度 (°C)
- T_0: 热力学临界温度 = 560°C
- φ: 裂变速率 (cm⁻³s⁻¹)
- A, n: 材料常数

### 简化工程公式 (U-10Mo)

```
T_stable ≈ 560 - 0.5 × log₁₀(φ / 10¹⁴)  [°C]
```

**计算示例**:
- φ = 2×10¹⁴ → T = 545°C
- φ = 4×10¹⁴ → T = 530°C
- φ = 6×10¹⁴ → T = 521°C

## 📊 RERTR验证数据

| 裂变速率 (×10¹⁴) | 温度 (°C) | γ相状态 |
|-------------------|-----------|---------|
| 2-3 | 250-400 | 稳定 |
| 4-5 | 200-350 | 稳定 |
| 5-6 | 150-300 | 部分分解 |

**关键发现**: 裂变速率增加1×10¹⁴，稳定温度降低10-20°C

---

## 🔍 适用范围

- 温度: 150-600°C
- 裂变速率: 10¹³-10¹⁵ cm⁻³s⁻¹
- Mo含量: 6-12 wt.%
