# 辐照UMo燃料中缺陷团簇与非平衡气泡耦合生长——团簇动力学与相场模型

## Metadata

## Overview

本文首次开发了耦合团簇动力学与相场模型的中尺度模拟框架，用于研究辐照下U-10wt%Mo合金（UMo）重结晶区内缺陷团簇与非平衡气体气泡的耦合生长行为。辐照实验表明，重结晶后气泡肿胀动力学远快于重结晶前，其根本原因在于晶粒细化（从微米级粗晶至200-300nm细晶）后气体气泡超晶格坍塌、基体应力释放，间隙原子团簇化倾向增强。

模型中，团簇动力学描述空位、间隙原子及其团簇的产生、扩散、复合、汇聚和发射过程；相场模型基于Kim-Kim-Suzuki（KKS）双平衡框架描述非平衡气泡的形核与生长，引入气泡内Xe浓度演化方程以描述从过压气泡到空洞状气泡的转变。两模型通过缺陷浓度和气泡结构的实时信息交换实现双向耦合。

通过系统参数化研究，发现：(1) 增加缺陷产生率、间隙原子发射率和团簇化率均加速肿胀，但仅团簇化效应可将模拟结果与实验数据对齐；(2) 晶界吸收导致的晶粒长大相关肿胀可忽略；(3) **间隙原子团簇化是重结晶区快速气泡肿胀的关键物理机制**。最佳参数组合为：G=3000（每裂变存活Frenkel对数）、$Y^{s}_{i,gb}=1.0$、$Y^{e}_{i,bb}=0.08$、$Z^0_{l_j}=2.4$。模拟预测的气泡密度、平均尺寸（~160nm）及Xe/空位比（约1:10）与实验数据合理一致。

## Key Equations

### 1. 缺陷浓度演化方程（团簇动力学核心方程）

$$\frac{\partial C_i}{\partial t} = D_i\nabla^2 C_i + \nabla\cdot\left(\frac{D_i C_i}{k_BT}\nabla U_i\right) + \dot{G}_i - \alpha C_V C_I + \sum_{l_j}K_{il_j}C_{l_j} - \sum_{l_j}\gamma_{il_j}C_{l_j} - \dot{S}_{i,def}C_i + \dot{\xi}_{i,def} - \rho_{dis}K_{i,dis}C_i$$

- $C_i(x,t)$：可动单缺陷（空位V或间隙原子I）浓度
- $D_i$：缺陷扩散系数
- $U_i$：缺陷在汇处的化学势
- $\dot{G}_i$：缺陷产生率
- $\alpha$：空位-间隙复合速率常数，$\alpha = Z_{iv}(D_I + D_V)/a^2$
- $K_{il_j}$：团簇$ml_j$对缺陷$i$的捕获系数
- $\gamma_{il_j}$：团簇$ml_j$对缺陷$i$的发射系数
- $\dot{S}_{i,def}$：缺陷在汇处的吸收速率
- $\dot{\xi}_{i,def}$：缺陷从汇处的发射速率

### 2. 气泡肿胀计算

$$S_{bb} = \frac{1}{V}\int \chi(\mathbf{x},t) dV$$

其中$\chi(\mathbf{x},t)$为相场序参量，$\chi=1$表示气泡内部，$\chi=0$表示基体。

### 3. 相场自由能（KKS模型）

$$F(C^g_V, \chi) = \int \left[f(\chi, C^g_V) + \frac{\kappa}{2}|\nabla\chi|^2\right] dV$$

$$f(\chi, C^g_V) = h(\chi)f^m(c^m_V) + (1-h(\chi))f^b(c^b_V) + wg(\chi)$$

- $h(\chi)=p(\chi)=\chi^3(10-15\chi+6\chi^2)$
- $g(\chi)=30\chi^2(1-\chi)^2$：双势阱函数
- $w$：势阱高度，$\kappa$：梯度能系数
- $f^m$、$f^b$：基体和气泡相的体自由能密度

### 4. Frenkel对产生率

$$\dot{G}_I = \dot{G}_V = G\dot{f}_0$$

- $G$：每次裂变存活的Frenkel对数（~1000-3000）
- $\dot{f}_0$：裂变率（$3.29\times10^{22}$ fissions/s/m³）
- NRT公式估计总Frenkel对~14825/裂变，但存活率仅~20%

### 5. 缺陷团簇捕获系数（归一化）

$$K^*_{il_j}(m) = K_{il_j}(m)/K^0_{il_j}(m) = Z_{il_j}/Z^0_{l_j}$$

- $Z^0_{l_j}$：团簇偏置系数的缩放值，$Z^0_{l_j}=2.4$时可拟合实验数据
- 间隙原子团簇化速率增大是快速肿胀的关键

## Physical Mechanisms

### 重结晶区快速肿胀机制
1. **超晶格坍塌**：重结晶使晶粒从微米级细化至200-300nm，纳米气泡fcc超晶格坍塌，基体压缩应力释放
2. **间隙原子团簇化**：应力释放使间隙原子团簇化在能量上有利，减少与空位的复合，更多空位可用于气泡生长
3. **非平衡气泡演化**：气泡内Xe浓度远低于平衡值（0.45），随裂变密度增加趋向空洞状（Xe/空位比~1:10）
4. **缺陷-气泡耦合**：气泡作为动态汇，其演化改变缺陷浓度场，形成正反馈加速生长

### 模型创新点
- 首次将微结构依赖的团簇动力学与相场模型双向耦合
- 引入局部发射/吸收区概念，替代传统速率理论中均匀再分布假设
- 描述从过压气泡到空洞状气泡的连续转变
