| 功能 | 数量 | 说明 |

| 代理 | 15+ | planner, architect, code-reviewer, security-reviewer, etc. |

#### 新增 v2 功能

**1. 持续学习 v2**
- `/instinct-status` - 查看学习的直觉（带置信度）
- `/instinct-import <file>` - 导入直觉
- `/instinct-export` - 导出直觉
- `/evolve` - 将直觉聚类到技能中

**2. Go 语言支持**
- Go 代码审查 (`/go-review`)
- Go 构建错误解决 (`/go-build`)
- Go TDD 工作流 (`/go-test`)
- Go 模式和最佳实践

**3. 技能创建器**
- `/skill-create` - 从 git 历史生成技能

**4. 跨平台支持**
- 完整支持 Windows、macOS 和 Linux
- 所有钩子和脚本用 Node.js 重写
- 包管理器自动检测（npm, pnpm, yarn, bun）
